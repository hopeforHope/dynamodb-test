## 실행 방법
operation 폴더 안에 각 작업 별로 파일을 나눠서 보관해두었습니다.
터미널에 해당 작업 파일을 직접 실행함으로 확인할 수 있습니다.
```linux
해당 폴더 위치로 이동
cd src/operation

실행 : node [실행할 파일]
ex) node read-ops.ts
```
해당 파일 안에 있는 모든 코드가 실행되는 방식이니, 필요한 부분만 남기고 나머지를 코멘트 처리 해서 사용할 수 있습니다.
